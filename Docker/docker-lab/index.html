通过Docker部署项目

本文最后更新于：2023年8月24日 上午

  1. 查看服务器系统

  2. ubuntu20.04安装docker(略过，自带)

  3. 查看docker版本

  4. 将后端项目打包

  5. 创建一个目录(docker-project)，用于存放 Docker 配置和后端项目的 jar 包

  6. 中目录中创建Dockerfile

    ~$ vim Dockerfile

  7. 在 backend 目录中打开终端，并构建 Docker 镜像。运行以下命令来构建镜像

    docker build -t mgmt-project-image .

    ps: 可以先查看docker状态与现有镜像

  8. 运行容器

    $ sudo docker run -d -p 8088:8080 --name mgmt-project-container mgmt-project-image

    其中8080是服务原本的端口，8088是映射在外的端口
